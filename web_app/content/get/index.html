CentOS 6

rpm --import http://rex.linux-files.org/RPM-GPG-KEY-REXIFY-REPO.CENTOS6

cat >/etc/yum.repos.d/rex.repo <<EOF
[rex]
name=Fedora \$releasever - \$basearch - Rex Repository
baseurl=http://rex.linux-files.org/CentOS/\$releasever/rex/\$basearch/
enabled=1
EOF

yum install rex

Gentoo

There is an overly for portage. To use this overlay open the file /var/lib/layman/cache_*.xml and add the following snippet.

<repo priority="50" quality="experimental" status="unofficial">
    <name>rex</name>
    <description>overlay for using rex on gentoo</description>
    <homepage>http://rexify.org/</homepage>
    <owner>
      <email>jan.gehring@gmail.com</email>
      <name>Jan Gehring</name>
    </owner>
    <source type="git">git://github.com/krimdomu/gentoo-overlay.git</source>
</repo>

And execute

bash# layman -a rex
bash# emerge -av app-admin/rex
